# Калькулятор

Простой, но мощный калькулятор с научными функциями.

## Функции

### Базовые операции
- Сложение, вычитание, умножение, деление
- Работа с десятичными числами
- Очистка и удаление символов

### Научные функции
- Sin, Cos (в градусах)
- Возведение в степень
- Квадратный корень
- Округление (floor, ceil)
- Остаток от деления

### Память
- M+ (добавить в память)
- M- (вычесть из памяти)
- MR (восстановить из памяти)
- MC (очистить память)

### Управление
- Поддержка клавиатуры
- Обработка ошибок
- Ограничение длины дисплея

## Использование

Откройте `index.html` в браузере.

### Горячие клавиши
- Цифры 0-9: ввод чисел
- +, -, *, /: операции
- Enter или =: расчет
- Escape: очистка
- Backspace: удаление символа
